# 数据挖掘基本概念
---
* **数据挖掘步骤**：
> - [ ] **挖掘准备**：定义挖掘问题，评估挖掘可行性
> - [ ] **数据清理**：消除噪声和不一致数据
> - [ ] **数据集成**：多种数据源组合在一起
> - [ ] **数据选择**：从数据库中检索与分析任务相关的数据
> - [ ] **数据变换**：数据变换或统一成适合挖掘的形式，如通过汇总或聚集操作
> - [ ] **数据挖掘**：基本步骤，使用只能方法提取数据模型
> - [ ] **模式评估**：根据某种选取度量，识别表示知识的真正有趣的模式
> - [ ] **知识表示**：用可视化和知识表示拘束，向用户提供挖掘的知识

* **数据挖掘对象**：原则上可以是任何信息存储类型中的数据包括
> * 关系型数据库
> * 数据仓库
> * 事务数据库
> * 面向对象数据库
> * 空间数据库
> * 文本数据库
> * 多媒体数据库
> * 时间序列数据库

* **数据挖掘方式**
> * 概念/类描述：用概念或类来描述数据特征和区分
> * 关联分析：以关联规则形式描述数据间的关系
> * 分类和预测：找出描述并区分数据类或概念的模型（或函数），利用模型预测类标记未知挖掘对象
> * 聚类分析：使得同一类中对象具有高相似性
> * 孤立点分析：与大多数对象相位或特征不一致的对象
> * 时序数据和序列数据分析：对具有时序和序列特征的数据进行演变分析，发现演变趋势模型

## 数据清理
> `在数据中消除错误和不一致`
### 数据清理任务
> **空值处理**
> > 1. 控制较少时，直接删除（忽略元组）
> > 2. 均值替换法、同类均值替换
> > 3. 专家经验法
> > 4. 猜测法
> > 5. 回归分析法
> > 6. 数据挖掘法
> 
> **噪声和错误数据处理**
> > 1. 平滑数据
> > 2. 小规模数据，手工去除或纠正
> > 3. 大规模数据，则应：
> > > * 定义错误类型或模式
> > > * 编制识别错误类型或模式的程序
> > > * 搜寻并识别错误实例
> > > * 去除或就在发现的错误
> 
> **不一致数据处理**
> > * 定义数据一致化标准
> > * 编制数据一致化标准程序
> > * 搜寻并识别不一致数据实例
> > * 纠正发现的不一致性错误

### 数据集成
> `将多个数据源中的数据合并并存放在一个统一的数据存储中`
> **注：**1. 数据源包括数据库、数据立方、文本文件等；2. 数据存储通常指数据仓库或数据库
### 数据集成方法
> 1. **利用数据库提供商提供的附加在数据库系统中的数据转换服务功能来实现多数据源的数据集成**
> 2. **自我分析并编写数据格式转换软件，实现数据集成**
### 数据集成的问题及解决方法
> 1. **实体识别问题**：不同数据库中id是否指同一个实体
> > `解决方法：`通过元数据提供的信息来解决
> 2. **冗余问题**：一个属性可由另一个或几个属性“表出”，则该属性冗余
> > `解决方法：`相关分析法
> 3. **数据值冲突的检测预处理**：由于表示、比例、编码等的不同，现实世界中的统一实体，在不同数据源中的属性值可能不同，如同样的价格但因货币不同导致数据不同。
> > `解决方法：`多靠人工分析发现冲突，并手工解决或编程自动解决

## 数据选择
> `在对挖掘任务和数据本身内容理解的基础上，寻找对挖掘效果有促进作用的数据，以缩减数据规模，减少挖掘计算的时间和空间开销。也称数据取样或数据简化`
### 数据选择方式
> * 数据域（属性、字段或列）选择
> * 元组（记录或行）选择
### 选择方法
> 依赖领域专家经验

---
## 数据变换
> `将数据变换成适合挖掘的形式`
### 数据变换方法
> 1. **数据离散化法**：将连续型数据一句离散化函数转为离散化数据
> 2. **变量（属性）构造**：通过构造新变量，提高原始数据挖掘效果
> 3. **拆分数据**：依据业务需求对数据项进行分解
> 4. **格式变换**：将一种数据格式变换成另一种同类的数据格式
> 5. **规范化方法**：将数据按比例缩放，使之落入一个小的特定的区域

## 数据归约
> `将初始数据集转换到某种更加紧凑的形式而又不丢失有意义的语义信息的过程。也叫数据缩减`
### 数据归约方法
> 1. **数据聚集**：对数据按语义逻辑进行汇总的操作。通常是对数据立方体进行上卷操作后得到的数据集。![Alt text](./1590638054267.png)
> 2. **维归约**：通过删除不相关数据属性来减少数据量
> > 常用贪心算法选择，具体有：逐步向前选择算法、逐步向后删除算法、向前选择和向后删除结合算法
> 
> 3. **属性值归约**：以少量的属性新数据代替大量的属性元数据
> > * 连续值离散化
> > * 直方图法
> > * 聚类：用聚类质心替换实际数据
> 
> 4. **选样**
> 简单n个不放回
> 简单n个放回
> > 聚类选用
> > 分层选样
