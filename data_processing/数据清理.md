# 数据清理
---
* **数据挖掘步骤**：
> - [ ] **挖掘准备**：定义挖掘问题，评估挖掘可行性
> - [ ] **数据清理**：消除噪声和不一致数据
> - [ ] **数据集成**：多种数据源组合在一起
> - [ ] **数据选择**：从数据库中检索与分析任务相关的数据
> - [ ] **数据变换**：数据变换或统一成适合挖掘的形式，如通过汇总或聚集操作
> - [ ] **数据挖掘**：基本步骤，使用只能方法提取数据模型
> - [ ] **模式评估**：根据某种选取度量，识别表示知识的真正有趣的模式
> - [ ] **知识表示**：用可视化和知识表示拘束，向用户提供挖掘的知识
